Proyecto 1: Análisis de Redes Sociales
Detección de Componentes Fuertemente Conectados usando Algoritmo de Kosaraju

Integrantes: [  Juan Ferreira C.I 31.804.073
		Alejandro Simanca C.I 30.911.678]
Repositorio GitHub: [https://github.com/DinkyVzla/SocialMedia]

Descripción: Aplicación Java con interfaz gráfica para analizar redes sociales
mediante grafos dirigidos. Incluye carga/guardado de archivos, modificación
de usuarios/relaciones, detección de comunidades y visualización gráfica.

Requerimientos cumplidos:
-Cargar archivo con JFileChooser
-Modificar grafo (agregar/eliminar usuarios y relaciones)
-Guardar cambios en archivo
-Mostrar grafo con GraphStream
-Detectar componentes fuertemente conectados (Kosaraju)
-Interfaz gráfica completa
-Documentación Javadoc
-Diagrama de clases

Tecnologías: Java, Swing, GraphStream
